# 🌦️ Weather App

## 📌 Description
A Python project that shows real-time weather information using the OpenWeatherMap API.

## 🚀 Features
- Live weather updates  
- City-based search  
- Temperature & humidity display  

## 🛠️ Tech Used
- Python  
- Requests library  
- OpenWeatherMap API  

## ▶️ Run
python weather.py

## 🎯 Purpose
Helps beginners learn API integration and JSON handling.
